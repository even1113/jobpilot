# Supervisor Check Prompt

Read first:

- `standards/README.md`
- `quality/definition-of-ready.md`
- `quality/definition-of-done.md`
- `quality/human-handoff.md`
- Related `US-xxx` requirement

Decide whether the work is acceptable.

Check:

- Does it satisfy the US acceptance criteria?
- Did implementation drift from the prototype?
- Were tests meaningful?
- Are skipped checks justified?
- Is human handoff required?
- Is Git traceability correct?

Return:

- Decision: accept, request changes, or human handoff.
- Reasons.
- Required next actions.
