# Code Review Prompt

Read first:

- `standards/README.md`
- `standards/git-standard.md`
- Related `US-xxx` requirement
- Test output or verification notes

Review in this order:

- Bugs and behavioral regressions.
- Missing tests.
- Requirement or prototype mismatch.
- Safety, privacy, and platform compliance issues.
- Git and US traceability.

Return findings first, ordered by severity, with file and line references when available.
