# Development Prompts

Use these prompts when implementing a formal requirement.

Required context:

- `standards/README.md`
- `standards/development-standard.md`
- `standards/testing-standard.md`
- `standards/git-standard.md`
- Related `US-xxx` requirement
- Related prototype spec when UI is involved

Do not implement a feature without a US ID unless the task is explicitly infrastructure-only.
