# Human Handoff

Pause automated work and ask a human to decide when:

- Requirements conflict or the intended behavior is ambiguous.
- The same test or implementation attempt fails repeatedly.
- Codex stops making progress or loops on the same fix.
- Implementation would violate the prototype, PRD/TDD, or acceptance criteria.
- The change affects privacy, credentials, resumes, salary, job-search truthfulness, or platform automation safety.
- The system encounters captcha, platform risk control, login failure, or access restriction.
- A decision changes product positioning, monetization, or user trust.

## Handoff Report

Provide:

- Related `US-xxx`.
- What was attempted.
- Exact blocker.
- Options considered.
- Recommended next decision.
- Current repo state and tests run.
