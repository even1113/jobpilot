# Quality Gates

`quality/` defines when work is ready to start, ready to ship, and when a human should take over.

## Files

- `definition-of-ready.md`: requirements before development starts.
- `definition-of-done.md`: completion standard.
- `test-gate.md`: test expectations by work type.
- `human-handoff.md`: conditions for human review or intervention.
- `agent-runbook.md`: what to do when Codex or another agent gets stuck.
