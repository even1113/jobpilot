# Ideas

`ideas/` is the parking lot for JobPilot ideas before they become formal requirements.

An idea becomes a formal requirement only after it has enough clarity to receive a `US-xxx` ID in `requirements/backlog.md`.

## Flow

1. Capture raw idea with `idea-template.md`.
2. Clarify user, pain, value, and non-goals.
3. Decide whether to promote it to `requirements/`.
4. If promoted, assign a new `US-xxx` ID and link back to the idea.

## Rules

- Do not build directly from an idea file.
- Do not assign `US` IDs inside `ideas/`.
- Keep rejected or paused ideas for future review instead of deleting context.
