# UX Research

`research/ux/` stores benchmark research and design references before high-fidelity prototype work.

## Rules

- Do benchmark research before redesigning a core JobPilot workflow.
- Record what is useful, what is not appropriate, and how it applies to JobPilot.
- Keep screenshots or external references in `references/` when licensing and attribution allow it.
- Link research from the related `US-xxx` requirement and prototype file.

## Output

Use `benchmark-template.md` for each UX research pass.
